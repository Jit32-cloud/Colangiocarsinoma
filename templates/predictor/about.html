About This Project

Cholangiocarcinoma is a cancer that forms in the bile ducts. Early risk awareness can support timely medical evaluation.

This project demonstrates a full-stack Django web system with a frontend form, backend validation, machine learning risk prediction integration, and result reporting.

Technology Used

  • Django (backend + templates)
  • HTML, CSS, JavaScript (frontend)
  • scikit-learn + joblib (ML model integration)
  • SQLite (data storage)
